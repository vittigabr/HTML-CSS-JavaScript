HTML

HTML (HyperText Markup Language) é a linguagem padrão de marcação para criar a estrutura de páginas web, definindo elementos como títulos, parágrafos, imagens e links através de tags. Essencial para desenvolvimento front-end, ela organiza o conteúdo, enquanto o CSS cuida do visual e o JavaScript da interatividade.

CSS

CSS (Cascading Style Sheets) é a linguagem fundamental usada para estilizar e definir a apresentação de documentos HTML ou XML. Ela controla o layout, cores, fontes, espaçamentos e responsividade, separando o conteúdo (HTML) da aparência. CSS3, atualizado por módulos, é o padrão moderno para criar interfaces visuais atraentes.

JavaScript

JavaScript (JS) é uma linguagem de programação dinâmica, baseada em protótipos e de alto nível, essencial para criar interatividade em páginas web (front-end) e no servidor com Node.js. É amplamente utilizada, suportando estilos de programação funcional e orientada a objetos.
